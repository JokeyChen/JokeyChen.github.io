GradeSource Checker

A script to automatically check the update on GradeSource and notify users via emails. It uses BeautifulSoup to parse the HTML and logs the update to a file. Users can schedule the script to run every 15 minutes using Linux Crontab. They can also configure the program easily using a setup script.

View on GitHub

  • Tools: Python, Git
  • Date: March 2016

6 Degrees of Kevin Bacon
(academic)

A school project exploring the degrees of separation between Hollywood actors that act in the same movies. Based on given parameters, it finds the earliest year two actors collaborated. Given the huge list of actors parsed from the IMDB database, my teammate and I implement this program using graphs and unions, which dramatically reduces the running time.

  • Tools: C++, Git
  • Date: June 2016
